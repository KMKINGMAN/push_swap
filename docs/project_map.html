Push_Swap Project Map

Stack Operations (stack_operations.c)

create_stack()
Creates and initializes a new stack structure
Returns: Pointer to new stack or NULL if allocation fails
push(t_stack *stack, int value)
Adds a new element to the top of the stack
Parameters: stack pointer, integer value

Sorting Algorithms (sort_utils.c, sort_large.c)

sort_three(t_stack *stack_a)
Sorts three elements using optimal moves
Optimal solution in maximum 2 moves
sort_five(t_stack *stack_a, t_stack *stack_b)
Sorts five elements using auxiliary stack
Uses find_min strategy
sort_large_stack(t_stack *stack_a, t_stack *stack_b)
Implements radix sort for larger numbers
Efficient for large datasets

Basic Operations (operations.c)

sa/sb(t_stack *stack, int print)
Swaps first two elements of stack A/B
pa/pb(t_stack *dest, t_stack *src)
Pushes top element from one stack to another

Rotate Operations (rotate_operations.c)

ra/rb(t_stack *stack, int print)
Shifts up all elements by 1
rra/rrb(t_stack *stack, int print)
Shifts down all elements by 1

Input Parsing (parser_utils.c)

parse_arguments(int argc, char **argv, t_stack *stack)
Processes and validates command line arguments
Handles both space-separated and individual number arguments

Error Handling (error_utils.c)

clean_exit(t_stack *stack_a, t_stack *stack_b, int error)
Handles program termination and cleanup
